About

The collaborative education analyzer is a unique real-time data collection and visualization tool for the immersive learning game City Settlers. The data collection tool will collect real-time voice data from one of two sources: 1) A Seeed Studio Core 2.0 6-Channel microphone; or 2) Multichannel voice capture from a discord server. The tool isolates the individual participants’ speech captured from the sources (both sources have this capability natively, but the code for it will need to be customized for the task), and sending them to a dedicated server. Once stored, the individuals’ speech will be processed using Natural Language Processing strategies to analyze students' performaces.
